# My Portfolio

Welcome to my personal portfolio! This repository contains the source code for my website where I showcase my work, skills, and professional journey.


### 💡 About This Project
This portfolio was crafted with **"Vibe Coding"**—a seamless fusion of intuitive development and deep technical execution. Leveraging efficient knowledge of **Node.js, CSS, and AI-driven workflows**, this project represents a commitment to building a meaningful existence in the **AI era**.

## 🚀 Technologies Used
* **Framework:** Next.js
* **Styling:** Tailwind CSS
* **Deployment:** Vercel
* **Animation:** Framer Motion
* **Core Logic:** Node.js
